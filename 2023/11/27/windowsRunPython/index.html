windows通过开机自动运行python脚本以发送邮件

November 27, 2023 638

运行环境:

Windows11,Python 3.11

创建 Python 脚本：

首先，你需要编写一个 Python 脚本。你可以使用文本编辑器（如Notepad++、VS Code等）编写你的脚本，确保脚本的路径中不包含中文或空格，以避免可能的问题。保存脚本并确保它能够在命令行中运行。

打开任务计划程序：

按下 Win + R 打开“运行”对话框。
输入 taskschd.msc 并按回车。

创建基本任务：

在任务计划程序中，选择 “创建基本任务”。
按照向导的提示，输入任务的名称和描述。
Alt text

PS:
笔者在这里建议新建一个文件夹，方便后续管理。

触发器：

选择 “启动时” 作为触发器，并按照提示设置触发任务的条件。

操作：

选择 “启动程序” 作为操作。

“程序或脚本”选择python路径。

“添加脚本”输入python脚本的路径。

完成：

完成向导并保存任务。

现在，你的 Python 脚本应该会在系统开机时自动运行。请注意，确保在任务计划程序中正确配置了 Python 的环境变量，以便它能够找到解释器。如果使用了虚拟环境，请在批处理文件中激活虚拟环境。
