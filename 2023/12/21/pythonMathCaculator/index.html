python自动导入库以进行科学计算

December 21, 2023 845

运行环境:

Windows11,Python 3.11

首先，确保 .pythonrc 文件位于你的用户主目录下（对于 Windows 用户，通常是 C:\Users\你的用户名）。然后，设置 PYTHONSTARTUP 环境变量以指向这个文件。在 Windows 上，可以按照以下步骤操作：

  1. 打开系统环境变量设置。你可以右键点击“此电脑”（或“计算机”），选择“属性”，然后点击“高级系统设置”。
  2. 在弹出的窗口中，点击“环境变量”按钮。
  3. 在“用户变量”部分，点击“新建”按钮。
  4. 在“变量名”中输入 PYTHONSTARTUP，在“变量值”中输入 .pythonrc 文件的完整路径（例如，C:\Users\你的用户名\.pythonrc）。
  5. 点击“确定”保存设置。

现在，当你启动 Python 解释器时，它应该会自动执行 .pythonrc 文件中的初始化代码。

然后退出 Python 解释器，重新启动它，看看是否加载了你的初始化配置。
